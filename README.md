# Simple Wallet

Simple Wallet is an Android app that is a virtual wallet

For the bills 1, 5, 10, 20, 50, 100, you may add them to your digital wallet. Your wallet will remember your individual bills, and totals. 

Consult the python example ([main.py](https://github.com/firebadnofire/simplewallet/blob/main/main.py)), tested on MacOS
